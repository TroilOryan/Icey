## 适配平台

- [x] Android
- [ ] iOS
- [ ] Pad
- [ ] Windows
- [ ] Linux

## feat

- [x] 检索本地音乐
- [x] 读取本地音乐标签,例如歌名、艺术家以及歌词
- [x] 播放模式
- [x] 播放记忆
- [x] 封面取色
- [x] 多种播放封面
- [x] 卡拉OK歌词
- [x] 自定义界面
